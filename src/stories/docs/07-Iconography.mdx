## Usage guidelines

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: 16, marginBottom: 48 }}>
<div style={{ padding: "20px", border: "1px solid var(--base-color-green-200, #BBF7D0)", borderRadius: 10, background: "var(--base-color-green-100, #DCFCE7)" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-green-800, #166534)", margin: "0 0 10px" }}>✓ Do</p>
  <ul style={{ fontSize: 13, lineHeight: 1.8, color: "var(--base-color-green-800, #166534)", margin: 0, paddingLeft: 18 }}>
    <li>Import from <code>lucide-react</code> only</li>
    <li>Use <code>size</code> prop, not CSS width/height</li>
    <li>Use colour tokens for icon colour, not raw hex</li>
    <li>Add <code>aria-label</code> to standalone (no-text) icons</li>
    <li>Pair icons with labels in navigation for accessibility</li>
  </ul>
</div>
<div style={{ padding: "20px", border: "1px solid var(--base-color-red-200, #FFB9BB)", borderRadius: 10, background: "var(--base-color-red-100, #FFE2E2)" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-red-600, #D21A26)", margin: "0 0 10px" }}>✗ Don't</p>
  <ul style={{ fontSize: 13, lineHeight: 1.8, color: "var(--base-color-red-600, #D21A26)", margin: 0, paddingLeft: 18 }}>
    <li>Don't mix Lucide with Material Icons, Heroicons, or others</li>
    <li>Don't use <code>fill</code> on Lucide icons — they are stroke-only</li>
    <li>Don't use more than one icon without a label in a row</li>
    <li>Don't use blue icons for decoration — only for active state</li>
    <li>Don't use icons smaller than 16px</li>
  </ul>
</div>
</div>
