## Column anatomy

Every DataTable column fits one of these types. Use the right cell pattern for each — it signals meaning at a glance.

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: 12, marginBottom: 48 }}>
  {[
    { type: "Identity cell", desc: "Primary + secondary line. Name bold, email/subtitle muted. Always the first column. Never truncate the name.", example: "Priya Sharma\npriya@acme.io" },
    { type: "Status badge", desc: "Use Badge with dot + semantic variant (success / warning / error). Never use plain text or raw colour for status.", example: "● Active\n○ Invited\n✕ Suspended" },
    { type: "Date / time", desc: "Relative time for recent events (2h ago, yesterday). Absolute date for historical records (Jan 2024). Use muted foreground.", example: "Just now\n3h ago\nJan 2024" },
    { type: "Numeric", desc: "Right-align all numbers. Use tabular-nums for alignment. Progress bars for values with a known max. Plain text for arbitrary counts.", example: "94\n12 projects\n↑ 8.2%" },
    { type: "Actions cell", desc: "Always last column. Use DropdownMenu with MoreHorizontal trigger for 3+ actions. Destructive actions below a separator, in destructive colour.", example: "⋯ → Edit / Change role / — / Remove" },
    { type: "Boolean / toggle", desc: "Use a read-only Switch or a checkmark icon. Never raw 'true/false' or 'yes/no' text.", example: "✓ Enabled\n— Disabled" },
  ].map(({ type, desc, example }) => (
    <div key={type} style={{ border: "1px solid var(--base-color-gray-200, #E3E3E0)", borderRadius: 10, padding: "14px 16px", background: "white" }}>

      <p style={{ fontSize: 12, color: "var(--base-color-gray-600, #6D6D6D)", lineHeight: 1.5, margin: "0 0 8px" }}>{desc}</p>
      <pre style={{ fontSize: 11, color: "var(--base-color-blue-800, #1518A6)", background: "var(--base-color-gray-75, #F5F5F3)", borderRadius: 6, padding: "6px 8px", margin: 0, fontFamily: "monospace", whiteSpace: "pre-wrap" }}>{example}</pre>
    </div>
  ))}

## The four data states

Every table or list has four states. All four must be designed. Missing states get built ad-hoc.

<div style={{ display: "grid", gridTemplateColumns: "repeat(4, 1fr)", gap: 12, marginBottom: 16 }}>
  {[
    { state: "Loading", colour: "#E0E7FF", textColour: "#1518A6", desc: "Show Skeleton rows matching the real column layout. Never show a spinner over the whole table — it collapses the layout.", rule: "Match column count and approximate row height" },
    { state: "Empty", colour: "var(--base-color-gray-75, #F5F5F3)", textColour: "var(--base-color-gray-700, #464646)", desc: "Icon (48px) + specific heading + supporting sentence + optional CTA. See Form Patterns / Empty States for copy rules.", rule: "Name what's missing, don't say 'No data found'" },
    { state: "Error", colour: "var(--alias-color-feedback-error-bg, #FFE9E5)", textColour: "var(--alias-color-feedback-error-fg, #FF4E2A)", desc: "Show Alert (error variant) above or in place of the table. Always give a retry action. Don't leave the user with a blank table.", rule: "Diagnose + explain + retry action" },
    { state: "Populated", colour: "white", textColour: "var(--base-color-gray-900, #222)", desc: "The default view. Show row count near the header. Enable sort and filter. Show pagination when rows exceed the page size.", rule: "Row count always visible" },
  ].map(({ state, colour, textColour, desc, rule }) => (

      <p style={{ fontSize: 12, color: "var(--base-color-gray-700, #464646)", lineHeight: 1.5, margin: "0 0 8px" }}>{desc}</p>
      <p style={{ fontSize: 11, fontStyle: "italic", color: "var(--base-color-gray-500, #909090)", margin: 0 }}>{rule}</p>

  ))}

## Pagination

  <div style={{ display: "grid", gridTemplateColumns: "160px 1fr", background: "var(--base-color-gray-100, #EEEEEC)", padding: "10px 20px", borderBottom: "1px solid var(--base-color-gray-200, #E3E3E0)" }}>
    {["Situation", "Rule"].map(h => <span key={h} style={{ fontSize: 11, fontWeight: 600, color: "#909090", textTransform: "uppercase", letterSpacing: "0.06em" }}>{h}</span>)}
  </div>
  {[
    { situation: "Default page size", rule: "10 rows. Offer 10 / 25 / 50 in a Select. Never default to 'All' — large datasets lock the browser." },
    { situation: "Pagination placement", rule: "Bottom of the table, full width. Left: row count (e.g. 'Showing 8 of 24 members'). Right: page size select + prev/next buttons." },
    { situation: "First / last page", rule: "Disable Prev on page 1, disable Next on last page. Don't hide them — disabled is better than absent." },
    { situation: "Page number buttons", rule: "Show max 3–5 page numbers. Use ellipsis (…) for gaps. Highlight current page with default (blue) button variant." },
    { situation: "After filtering", rule: "Reset to page 1. Update the row count to reflect filtered results.' Showing 3 of 24 members' not '0 of 24'." },
  ].map(({ situation, rule }, i) => (
    <div key={situation} style={{ display: "grid", gridTemplateColumns: "160px 1fr", padding: "12px 20px", borderBottom: i < 4 ? "1px solid var(--base-color-gray-200, #E3E3E0)" : "none", background: i % 2 === 0 ? "white" : "var(--base-color-gray-75)" }}>
      <span style={{ fontSize: 12, fontWeight: 600, color: "#222" }}>{situation}</span>
      <span style={{ fontSize: 13, color: "#464646", lineHeight: 1.5 }}>{rule}</span>
    </div>
  ))}
</div>
