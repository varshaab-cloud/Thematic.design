## Usage rules

<div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: 16 }}>

<div style={{ background: "var(--base-color-gray-75, #F5F5F3)", border: "1px solid var(--base-color-gray-200, #E3E3E0)", borderRadius: 10, padding: "20px 20px 16px" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-gray-900, #222)", margin: "0 0 12px" }}>Hierarchy through weight + size</p>
  <p style={{ fontSize: 13, color: "var(--base-color-gray-600, #6D6D6D)", margin: 0, lineHeight: 1.6 }}>Use weight and size contrast to signal importance. Reserve colour changes for semantic meaning only — not for decorative hierarchy.</p>
</div>

<div style={{ background: "var(--base-color-gray-75, #F5F5F3)", border: "1px solid var(--base-color-gray-200, #E3E3E0)", borderRadius: 10, padding: "20px 20px 16px" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-gray-900, #222)", margin: "0 0 12px" }}>Metric numbers</p>
  <p style={{ fontSize: 13, color: "var(--base-color-gray-600, #6D6D6D)", margin: 0, lineHeight: 1.6 }}>KPIs: 700 weight at 24–36px. Unit labels drop to Body Text 2. This size contrast creates immediate scanability across a dashboard.</p>
</div>

<div style={{ background: "var(--base-color-gray-75, #F5F5F3)", border: "1px solid var(--base-color-gray-200, #E3E3E0)", borderRadius: 10, padding: "20px 20px 16px" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-gray-900, #222)", margin: "0 0 12px" }}>Muted text = Gray 500</p>
  <p style={{ fontSize: 13, color: "var(--base-color-gray-600, #6D6D6D)", margin: 0, lineHeight: 1.6 }}>Helper text, timestamps, and secondary labels use gray-500 (#909090). It is still readable but clearly secondary — never use gray-400 for text.</p>
</div>

<div style={{ background: "var(--base-color-gray-75, #F5F5F3)", border: "1px solid var(--base-color-gray-200, #E3E3E0)", borderRadius: 10, padding: "20px 20px 16px" }}>
  <p style={{ fontSize: 13, fontWeight: 600, color: "var(--base-color-gray-900, #222)", margin: "0 0 12px" }}>Section labels = Caption 2</p>
  <p style={{ fontSize: 13, color: "var(--base-color-gray-600, #6D6D6D)", margin: 0, lineHeight: 1.6 }}>0.625rem / 600 / uppercase / tracking-widest. Used for sidebar section titles, table column groups, form section dividers.</p>
</div>
