## Alert — severity variants

Alert has five variants. Each maps to a specific situation. The variant controls colour, border, and icon colour — all via alias tokens.

## AlertDialog — confirmation gates

Use `AlertDialog` only when an action is **irreversible or removes data permanently**. It blocks all other interaction until the user responds — which is why it should be rare.
